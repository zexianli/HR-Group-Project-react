## Running Tests
### Run all tests (watch mode)
```
npm test
```
### Run once (CI mode)
```
npm run test:run
```
### Run a single feature
```
npx vitest src/features/documents
```
### Run a single test file
```
npx vitest src/features/visa/__tests__/visaSlice.test.js
```
